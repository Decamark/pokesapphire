StringCopy

u8 *StringCopy(u8 *dest, const u8 *src)
{
    while (*src != EOS)
    {
        *dest = *src;
        dest++;
        src++;
    }

    *dest = EOS;
    return dest;
}

This function can cause an overflow as it copies src to dest until it finds EOS. The following is an example:
